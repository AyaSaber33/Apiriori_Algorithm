
# 🧠 Apriori Algorithm (From Scratch - Python)

Implementation of the **Apriori Algorithm** in Python for **frequent itemset mining** and **association rule generation**.

---

## 📌 Key Idea
Apriori uses the principle:
> If an itemset is frequent, all its subsets must also be frequent.

---

## ⚙️ Features
- Pure Python (no external libraries)
- Frequent itemset generation
- Support-based pruning
- Association rules with confidence
